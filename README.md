# BlackjackOOP

Console Blackjack game built as part of my Python learning process.

Description

This is a simple Blackjack implementation using Object-Oriented Programming.
The goal of the project is to practice structuring code with classes and managing game logic.

Features

- Card system (name, suit, value)
- Hand management (storing cards, calculating score)
- Random card generation
- Main game loop (hit / stand)
- Bust detection (>21)
- Basic win condition (21)
- Replay system

Project Structure

- "Card" — represents a single card
- "Hand" — stores cards and calculates total value
- "Game" — controls the game flow and player interaction

How to Run

1. Make sure Python is installed
2. Run the file:

python main.py
